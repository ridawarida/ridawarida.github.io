Works on Machine Learning

  • From Leaves to Fruits: A Computer Vision Framework for Early-Stage Mango Quality Prediction and Mango Classification: Built a non-destructive, post-harvest computer vision framework using deep learning architectures (EfficientNet-B7, DenseNet-121, ConvNeXt Tiny, and Swin Transformer V2 Tiny) and three hybrid clas- sifiers (Random Forest, SVM, Logistic Regression) to classify mango varieties and predicts sweetness profile from their leaf images, addressing challenges of data scarcity and high visual similarity among selected cultivars.
  • Predictive Analysis of Diabetes Using Machine Learning: Developed a machine learning model to predict the likelihood of diabetes in patients based on a set of health-related features. Several supervised learning models, including Logistic Regression, Decision Tree, and a Neural Network built using TensorFlow, were implemented and compared with the aim of determining which approach performs best in terms of classification accuracy and reliability.
  • Machine Learning Analysis of Music’s Impact on Mental Health Trends: Evaluated four classification models—Random Forest, Gradient Boosting, XGBoost, and Logistic Regression—to predict music-induced mental health outcomes, categorizing effects into Worsen, No effect, and Improve using a dataset of 723 self-reported samples,

Works on Robotics

  • PathGuard: Discrete Logic-Driven Obstacle Avoider: Developed an au- tonomous robot car developed using fundamental hardware logic principles without the use of a microcontroller. The system detects obstacles in its path using infrared (IR) sen- sors and navigates by steering in the opposite direction, driven entirely by discrete transistor-based NOT gates and a dual H-bridge motor driver. The motivation behind this work stems from the long-term goal of developing assistive technologies for the elderly and physically challenged. The physical requirement for such assistive robots is the abil- ity to navigate indoor environments autonomously, avoiding common household obstacles.

Works on Game Development

  • "Red-light green-light": Inspired by the popular game from the Netflix series "Squid Game", I, along with my project partner, developed a 3D game on OpenGL where players must navigate a course while avoiding detection by a "doll" that turns around at random intervals. The game features multiple levels of increasing difficulty, with various game modes and cheat systems to enhance gameplay. The objective is to reach the finish line without being caught, providing an engaging and challenging gaming experience.
